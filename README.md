# Вебкузница

Сайты для местного бизнеса — ресторанов, салонов красоты, фитнес-клубов.

## Что это

Каталог демо-сайтов, которые мы делаем индивидуально под характер каждого
заведения. Не штампуем по шаблону — куём руками.

## Сегменты

- **Рестораны** ✅ активно — `/restaurants/`
- **Салоны красоты** — готовится
- **Фитнес** — готовится

- [Главная](https://viktorialiubitskaya-coder.github.io/vebkuznitsa/)
- [Каталог ресторанов](https://viktorialiubitskaya-coder.github.io/vebkuznitsa/restaurants/)
- [«В Коломне-Есть!» — ресторан в Коломне](https://viktorialiubitskaya-coder.github.io/vebkuznitsa/restaurants/v-kolomne-est/)

## Stack

Pure HTML + Tailwind CSS via CDN. No build step — static files served via GitHub Pages.
